More information about (FIP)

Project Main Points

ASSIGNMENT DESCRIPTION

Each student is required to create three (3) additional functioning mobile and three (3) additional functioning desktop web pages for the Couples Resort website based on the HTML/CSS templates provided in class. All 3(x2) pages must correspond to a link in the main menu (Rooms, Packages, Dining, Spa Activities, Reservations). Additional photos, text, information, functionality, usability can be added to supplement your custom designed page(s), if you feel that something might be missing from the original page of the Couples Resort website.

ASSIGNMENT REQUIREMENTS

Each student must create three mobile (412px) and three Desktop (1280px) wireframe and design for the Couples Resort website. You CAN NOT USE the same page or content from Homework 1 and 2 or the in-class work on the homepage. Grades in Design & Image are based on the design work for you Couples Resort pages, not the code. Grades will be given based on your working file of the wireframe and prototyping tool you use to create your work. We will be using the Adobe XD software for our lessons and lesson files because it is provided to all students as part of the software licence at Fanshawe.

SUBMISSION REQUIREMENTS & DUE DATE

Projects must be submitted Week 14. Github repo: readme.md file, master branch, development branch(es) Name the branches appropriately per feature IE des.yourinitials.artwork, dev.yourinitials.script Submit the repo link via FOL dropbox DO NOT SUBMIT ZIPPED FILES, ONLY THE MASTER BRANCH WILL BE GRADED
